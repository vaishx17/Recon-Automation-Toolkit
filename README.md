# Recon Automation Toolkit

A Python-based reconnaissance automation toolkit developed to streamline information gathering and network enumeration tasks commonly performed during the initial stages of security assessments.

## Features

- IP Host Discovery
- TCP Port Scanning
- Subdomain Validation
- Phone Number Information Lookup
- Password Generation
- Wordlist Generation

## Technologies Used

- Python
- Socket Programming
- Requests Library
- OSINT Techniques

## Installation

```bash
git clone https://github.com/vaishx17/Recon-Automation-Toolkit.git
cd Recon-Automation-Toolkit
pip install -r requirements.txt
```

## Usage

```bash
python recon_tool.py
```

## Sample Output

The toolkit provides a menu-driven interface for executing reconnaissance and information-gathering tasks.
